How it works: Each URL is sent to a serverless function that launches a headless Chrome browser, navigates to the page, waits for it to fully render, then exports it as a PDF and streams it back to you. Works with any publicly accessible URL. For bulk downloads, all PDFs are packaged into a single ZIP file.
